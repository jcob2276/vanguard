# Kierunek — North Star i plany etapów

Ten folder (dawniej `KIERUNEK NAJWAŻNIEJSZE!/` w root) zawiera **najważniejsze dokumenty** określające kierunek, w którym zmierza Vanguard OS.

## Pliki w tym folderze

- `KIERUNEK.md` ← **North Star / Wersja 10/10** (to jest najważniejszy dokument)
- `ETAP_0_*` ← komplet dokumentów audytu, dualizmu, modelu danych i planu wdrożeniowego Etapu 0
- `ETAP_1_BACKLOG.md` ← pogłębiony backlog Etapu 1 (Pattern Memory + Anty-Self-Deception)
- `ETAP_1_RESEARCH_WZORCE_ISTNIEJACE_DANE.md` ← research: jakie realne wzorce da się już teraz wykryć z friction/p2/aggregates
- `ETAP_1_INTERFEJSY_WZORCE_PROPOZYCJE.md` ← konkretne pierwsze miejsca i formaty pokazywania wzorców użytkownikowi
- `ETAP_1_START.md` ← punkt wejścia do Etapu 1

## Główne dokumenty w repo (powiązane)

- [`../ROADMAP_V10.md`](../ROADMAP_V10.md) — kanoniczna roadmapa (pełna wersja)
