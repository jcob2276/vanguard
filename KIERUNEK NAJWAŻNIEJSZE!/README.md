- `KIERUNEK.md` ← **North Star / Wersja 10/10** (to jest najważniejszy dokument)

- `ETAP_1_BACKLOG.md` ← pogłębiony backlog Etapu 1 (Pattern Memory + Anty-Self-Deception)
